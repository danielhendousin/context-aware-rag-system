# Thesis Summary

## Title
Türkiye'deki Kamu Şirketlerinin Finansal Raporlarının LLM Modeli

## Project Goal
This thesis explores how large language models can interact with uploaded Turkish PDF documents through a Retrieval-Augmented Generation (RAG) pipeline.

## Main Components
- LangChain-based orchestration
- FAISS vector database
- PDF ingestion and chunking
- Prompt templates
- Output parsers
- Memory-aware conversation flow
- Context-aware retrieval for follow-up questions

## Evaluation Highlights
The thesis reports:
- More than 30 total questions tested
- 7 representative financial questions analyzed in detail
- Correct retrieval of values such as cash balances, gross profit, inflation rate, exchange rates, note-based figures, and inventory values
- Correct handling of numerical and table-based financial information in the analyzed examples
